# README - Script Mô phỏng Xe buýt (bus_sim.js)

Script này dùng để mô phỏng một xe buýt đang chạy và gửi tọa độ (GPS) tới máy chủ Socket.IO để test chức năng realtime.

## 1. Cài đặt

Script này yêu cầu Node.js và thư viện `socket.io-client`.

```bash
# Cài đặt thư viện
npm install socket.io-client
